AI-RAG-System

一个基于 RAG（Retrieval-Augmented Generation） 的本地知识问答系统。

该项目通过向量检索与大语言模型结合，实现对本地文档的智能问答和知识检索。

RAG 是一种将文档检索与语言模型生成结合的方法，可以显著减少模型幻觉并提高回答准确性。

项目功能

📄 文档加载：支持加载本地知识库文档

🔍 向量检索：将文本向量化并进行相似度搜索

🤖 RAG问答：结合检索结果生成智能回答

🌐 Web界面：通过网页进行交互式问答

⚡ 缓存机制：加速向量检索与响应速度

🧪 测试模块：包含 loader、chat、rag 等测试脚本

工作流程

加载本地文档数据

文本切分并生成向量

存入向量索引库

用户提问

检索相关文本

结合上下文生成回答
